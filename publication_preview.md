# Publication Preview — LJT-Homepage style

> Entries below are rendered exactly as the academic-pages Jekyll template
> would display them on the `/publications/` page, grouped by the
> `publication_category` keys defined in `_config.yml`
> (`manuscripts` → **Journal Articles**; `conferences` → **Conference Papers**),
> and sorted in reverse-chronological order within each group.
> The *title* is a relative link to the paper's permalink; the citation line
> reproduces the `citation` front-matter field with HTML entities decoded.
